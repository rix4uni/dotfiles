## Setup
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm --depth 1
mkdir -p ~/.config/tmux
wget -q https://raw.githubusercontent.com/rix4uni/dotfiles/refs/heads/main/tmux/tmux.conf -O ~/.config/tmux/tmux.conf
```

### For Termius
```
wget -q https://raw.githubusercontent.com/rix4uni/dotfiles/refs/heads/main/tmux/tmux-termius.conf -O ~/.tmux.conf
```
